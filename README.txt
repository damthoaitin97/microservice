CHÀO MỪNG ĐẾN VỚI ỨNG DỤNG
========================

Đây là một ứng dụng web được xây dựng bằng công nghệ microservice.

CÁC TÍNH NĂNG CHÍNH
------------------
1. Đăng ký và đăng nhập tài khoản
2. Quản lý sản phẩm
3. Tìm kiếm sản phẩm
4. Quản lý đơn hàng

YÊU CẦU HỆ THỐNG
--------------
- Windows 10/11 hoặc Mac OS
- 4GB RAM trở lên
- 10GB ổ cứng trống
- Kết nối internet

CÁCH CÀI ĐẶT VÀ CHẠY
-------------------
Vui lòng đọc file "HƯỚNG DẪN CÀI ĐẶT.txt" để biết chi tiết.

CÁC TRANG WEB CHÍNH
-----------------
1. Trang chủ: http://localhost:8080
2. Trang quản lý: http://localhost:8080/admin

LIÊN HỆ HỖ TRỢ
------------
Nếu bạn gặp bất kỳ vấn đề nào, vui lòng liên hệ:
- Email: [your-email]

LƯU Ý
-----
